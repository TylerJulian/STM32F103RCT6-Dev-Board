# STM32F103RCT6 Developement board

A stm32f103RCT6 Breakout board for Sooner Competitive Robotics and Personal use.

Contains PORTs for CAN-Bus protocol and a 6 pin molex power and signal connection.

## Components

1x STM32f103RCT6

1x CAN transceiver
